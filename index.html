Use these three properties to create a flexbox row layout.

.row{
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Flexbox Column

.column{
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

.grid{
    display: grid;
    width: 100%;
    grid-template-columns:
    repeat(12, 1fr);
}
